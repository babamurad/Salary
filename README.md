# Payroll System (Система расчета заработной платы)

Десктопное приложение для автоматизации кадрового учета и расчета заработной платы. Программа разработана на Delphi с использованием локальной базы данных SQLite, что делает её быстрой, переносимой и независимой от громоздких серверов баз данных.

## 🚀 Основные возможности

* **Управление персоналом:** Ведение карточек сотрудников (ФИО, должность, отдел, дата приема, оклад).
* **Многофирменный учет:** Возможность создавать новые чистые базы данных (`.db`) для разных компаний и мгновенно переключаться между ними.
* **Гибкие настройки:** Настраиваемые ставки налогов, отчислений и коэффициенты для больничных листов в зависимости от стажа.
* **Производственный календарь:** Учет рабочих дней и норм часов по месяцам и годам.
* **Расчет отпускных и больничных:** Автоматический расчет на основе истории начислений, стажа работы (до 5 лет, от 5 до 8 лет, свыше 8 лет) и количества дней.
* **Дашборд:** Наглядный главный экран с ключевыми метриками (количество активных сотрудников, месячный фонд окладов).
* **Защита данных:** Система "мягкого удаления" сотрудников (перевод в статус "Уволен"), чтобы не нарушать исторические расчеты в журналах начислений.

## 🛠 Технологии

* **Язык/Среда:** Delphi / RAD Studio
* **База данных:** SQLite
* **Компоненты доступа к данным:** FireDAC
